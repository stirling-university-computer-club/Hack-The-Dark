JavaScript allows the declaration of variables.

the code inside the script tag is

today = new Date() ;

creates a Date object, and stores it in the variable today.

window.document.write("<p><b>today's date is </b>" + today.getDate() + " " + (today.getMonth()+1) + " " + today.getYear()) ) ;

writes out the day, month, and year, by using methods in the Date object. window.document.write("<p><b>the time is</b>" + today.getHours() + ":" + today.getMinutes()) ;

writes out the time, using methods in the Date object. window.document.write("<p><b>this can also be written out as </b>" + today.toGMTString()) ;

Can write out the whole content of the Date object as a single string

the getDate() method of date returns the day of the month.

the getMonth() method returns the month as a number between 0 and 11 .

the getYear() returns the year (as a number since 1900).

the getHours() method returns the hour of the day, and the getMinutes() method returns the minutes of the time.

the toGMTString() takes the internal date format and returns a complete string of the date and time.
